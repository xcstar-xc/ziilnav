## 🚀 1. 快速开始：账号与同步
子了导航采用免注册、基于密钥 (Key) 的同步机制。
<img width="842" height="216" alt="image" src="https://github.com/user-attachments/assets/b9a53203-243d-429a-86d7-1d7513417418" />

### 创建/登录：
- 在首次访问时，您会看到“输入密钥”的提示。
- **任意输入一个您喜欢的字符串**（例如 `my-secret-space-2024`）作为您的主密钥。
- 点击“同步”，系统会自动为您创建一个新空间。
- **注意**：请牢记您的密钥！在任何新设备上输入相同的密钥，即可无缝同步您的所有数据。

### 退出登录：
- 点击侧边栏底部的“设置 / 已同步”按钮。
- 点击红色的“退出当前账户”即可清除本地缓存。

## 🔖 2. 书签管理
核心功能采用横向胶囊卡片设计，支持丰富的自定义选项。
<img width="810" height="336" alt="image" src="https://github.com/user-attachments/assets/3dd35456-03ae-4d41-b8d2-9ec2df59f42d" />

### 添加书签
- **入口**：
    - 电脑端：点击页面右上角的 `+ 添加` 按钮。
    - 手机端：点击书签列表第一个虚线框的 `+` 卡片。
- **智能获取**：
    - 输入网址后，系统会自动尝试获取网站图标 (Favicon)。
    - 系统会自动抓取网页标题（支持多级重试机制，确保准确率）。
- **图标选择**：
    - **自动获取**：默认选项，抓取目标网站图标。
    - **内置图标**：提供 100+ 个精选矢量图标。
    - **Emoji**：内置海量 Emoji 表情库，支持分类浏览。
    - **在线搜索**：特色功能！输入中文（如“微信”）或英文关键词，实时搜索海量开源图标库。
- **配色方案**：
    - 支持 18 种马卡龙流光渐变色（如蜜桃、薄荷、极光、日落等）。
    - 支持“自动配色”，根据网址哈希值自动分配颜色。

### 编辑与操作
- **编辑**：鼠标悬停在书签卡片上，点击右上角的 ✏️ 图标。
- **删除**：鼠标悬停在书签卡片上，点击右上角的 🗑️ 图标。
- **置顶**：点击 📌 图标，将常用网站固定在列表最上方。
- **拖拽排序**：按住书签卡片即可拖拽改变顺序。

## 📂 3. 分类管理 (所见即所得)
我们引入了类似 iOS 列表的“编辑模式”，让分类管理变得无比直观。
<img width="800" height="261" alt="image" src="https://github.com/user-attachments/assets/94abd824-6171-41b3-8e36-ec09e9302bce" />

### 一级菜单 (侧边栏)
- **进入编辑模式**：点击侧边栏标题旁的 “编辑” 文字按钮。
- **排序**：按住右侧的“三横线”手柄，上下拖拽调整分类顺序。
- **删除**：点击左侧的红色“减号”图标删除分类（注意：若分类下有内容，需先清空）。
- **新建**：在编辑模式下，列表底部会出现 “+ 添加分类” 按钮。
- **完成**：操作完毕后，点击 “完成” 保存状态。

### 二级菜单 (顶部栏)
- **进入编辑模式**：点击顶部栏右侧的 “编辑” 按钮。
- **排序**：直接拖拽胶囊标签进行左右排序。
- **管理**：
    - 点击胶囊上的 ✏️ 修改名称。
    - 点击胶囊左上角的 🔴 删除子分类。
- **新建**：编辑模式下，列表末尾会出现虚线的 “+ 添加” 按钮。

## 🤝 4. 访客模式 (分享功能)
想把导航分享给朋友，但怕被乱改？使用访客模式。
<img width="808" height="395" alt="image" src="https://github.com/user-attachments/assets/ad787980-e116-4589-bb94-ea85f1e12a5e" />

### 生成分享码：
1. 点击左下角“设置”，找到 “分享我的导航” 区域。
2. 您可以自定义一个分享密钥（如 `share-my-nav`），或使用系统生成的。
3. 点击“设置”保存，并复制密钥。

### 他人访问：
- 朋友在登录页输入这个 **分享密钥**。
- 系统会进入 “只读模式 (Guest View)”。
- **限制**：访客只能查看和点击链接，无法添加、编辑、删除、排序，也无法查看您的主密钥。

## 🧩 5. 实用小组件
主页顶部支持最多 4 个实用小组件，可在“设置”中自由开关。
<img width="807" height="141" alt="image" src="https://github.com/user-attachments/assets/def30ed4-549b-4d3c-8f95-4fae701d8f10" />

- **一言 (Hitokoto)**：
    - 每日一句灵感语录，支持点击刷新和复制。
- **待办事项 (Todo)**：
    - 极简的清单管理，支持勾选完成和删除。
- **临时便签 (Scratchpad)**：
    - 一个随手记的文本框，内容会自动同步保存。
- **倒计时 (Days Matter)**：
    - 设置一个重要日期（如春节、纪念日），显示剩余天数或已过去天数。

## ⚙️ 6. 个性化与数据
- **深色模式**：点击设置中的太阳/月亮图标切换，完美适配夜间浏览（荧光配色）。
- **搜索引擎**：点击搜索框左侧图标，可切换 Google、百度、Bilibili、知乎等搜索引擎。
- **布局切换**：支持“网格视图”（默认）和“列表视图”。
- **数据备份**：
    - **导出 JSON**：备份包含所有设置和数据的完整文件。
    - **导出 HTML**：导出通用浏览器书签文件。
    - **导入 HTML**：支持从 Chrome/Edge 等浏览器导入书签文件。

## ❓ 常见问题 (FAQ)

**Q: 我的数据存在哪里？**
A: 数据存储在云端数据库中，通过您的“主密钥”进行索引。请务必保管好您的密钥，它是找回数据的唯一凭证。

**Q: 为什么有些网站图标显示不出来？**
A: 部分网站有防抓取策略。您可以尝试使用“在线搜索”功能，手动搜索该网站的 Logo（如搜索 “bilibili”）来替换。

**Q: 忘记密钥怎么办？**
A: 由于系统采用匿名加密机制，无法找回丢失的密钥。建议您首次使用后立即在本地保存一份 JSON 备份，或将密钥记录在安全的地方。
